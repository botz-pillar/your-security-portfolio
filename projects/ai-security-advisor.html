AI Security Advisor

An AI-powered assistant that can answer questions about cloud security topics and about the projects, skills, and experience featured in this portfolio. Built with the Claude API.

Try It Below Back to Projects

What This Is

This AI advisor is a live, interactive feature built into the portfolio. It uses the Claude API to answer two kinds of questions: general cloud security topics (AWS, IAM, encryption, detection, architecture) and specific questions about the portfolio owner's projects, skills, and experience.

How It Works

  • Runs entirely in the browser — no backend server needed
  • Calls the Claude API directly using the portfolio owner's API key
  • System prompt is built dynamically from portfolio config and resume context
  • Scoped to cloud security and portfolio-related topics
  • Conversation resets on page refresh to keep API costs low

Why This Matters

This feature demonstrates practical AI integration: calling an API from client-side code, designing a scoped system prompt, handling streaming responses, and building a polished UI — all within a static site. It also gives visitors an interactive way to explore the portfolio beyond just reading pages.

Setup

To enable this feature, add your Anthropic API key to the aiAdvisorApiKey field in assets/js/main.js. You can also customize the resume context, welcome message, and sample questions from the same config file. See the Tier 3 section under "AI Security Advisor."
